DIALUX EVO: Eclairage d'une adiministration

Ce tutoriel présente les étapes essentielles pour réaliser une étude d’éclairage d’un bâtiment administratif avec DIALux Evo. L’objectif est de modéliser les locaux, de définir les besoins lumineux et de générer un rapport conforme aux normes en vigueur.

1. Préparer le projet

  • Collecter les plans architecturaux (format DWG ou PDF) : bureaux, couloirs, salles de réunion, hall d’accueil.
    Pour le coup moi j'ai téléchargé mes fichiers de plans sur un site bien utile qui dispose plusieurs modèles de fichier .dwg pour les plans de bâtiment. Vous pouvez utiliser le lien pour accéder au site ou télécharger mon fichier totalement apprêté pour l'usage dans un projet Dialux evo.
  • Identifier les niveaux d’éclairement requis selon les recommandations (par exemple 300 lux pour les zones de travail).
  • Ouvrir la version de DIALux Evo installée et créer un nouveau projet (Fichier > Nouveau projet).

2. Modéliser le bâtiment

  • Importer ou tracer le plan du bâtiment : Construction > Importer plan.
  • Définir les zones et pièces : bureaux, couloirs, salles techniques, etc.
  • Ajuster la hauteur sous plafond et les matériaux (couleurs, réflectances) pour une simulation réaliste.

3. Choisir et placer les luminaires

  • Sélectionner les luminaires certifiés dans la base de données des fabricants (fichiers IES ou LDT).
  • Placer les appareils selon les zones : éclairage direct pour les postes de travail, éclairage d’ambiance pour les circulations.
  • Définir la hauteur d’installation et vérifier l’espacement entre les luminaires.

4. Paramétrer les calculs

  • Accéder à Calcul > Paramètres et choisir l’éclairement moyen comme critère principal.
  • Fixer la grille de calcul (distance entre points, hauteur du plan de travail : 0,8 m typiquement).
  • Lancer le calcul photométrique (Calcul > Démarrer).

5. Analyser les résultats

  • Vérifier les niveaux d’éclairement en lux et l’uniformité (U0 > 0,6 recommandé pour les bureaux).
  • Consulter les faux-couleurs et les valeurs ponctuelles pour repérer les zones insuffisamment éclairées.

6. Générer le rapport d’étude

  • Dans Documentation > Rapport, choisir les sections nécessaires : plans, données de calcul, fiches techniques des luminaires.
  • Exporter le rapport en PDF pour remise au bureau d’études ou au client.

Conseils pratiques

  • Utiliser les plug-ins des fabricants pour obtenir les dernières références de luminaires.
  • En phase de conception, toujours vérifier la conformité avec les normes locales et internationales (ex. EN 12464-1).
  • Prévoir un éclairage de secours (issue de secours, balisage) si le projet l’exige.

Avec ces étapes, vous disposez d’une méthode complète pour réaliser un dimensionnement d’éclairage professionnel d’une administration grâce à DIALux Evo.
